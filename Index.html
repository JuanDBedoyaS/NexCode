Conceptos Fundamentales de Programación

Antes de aprender Python o Java es importante comprender los conceptos que utilizan todos los lenguajes de programación. Aquí encontrarás explicaciones sencillas y ejemplos prácticos.

📦 Variables

Una variable es un espacio donde se almacena información para usarla más adelante. Piensa en ella como una caja con una etiqueta.

Python

nombre = "Juan" edad = 25 print(nombre) print(edad)

Java

String nombre = "Juan"; int edad = 25; System.out.println(nombre); System.out.println(edad);

🖨️ print() / System.out.println()

Permite mostrar información en pantalla. Es una de las instrucciones más utilizadas cuando estamos aprendiendo.

Python

print("Hola Mundo")

Java

System.out.println("Hola Mundo");

⌨️ Entrada de Datos

Permite que el usuario escriba información desde el teclado.

Python

nombre = input("Ingrese su nombre: ") print(nombre)

Java

Scanner sc = new Scanner(System.in); String nombre = sc.nextLine(); System.out.println(nombre);

🔀 Condicional IF

Permite tomar decisiones. Si una condición es verdadera, se ejecuta un bloque de código.

Python

edad = 18 if edad >= 18: print("Es mayor de edad")

Java

int edad = 18; if(edad >= 18){ System.out.println("Es mayor de edad"); }

↔️ IF / ELSE

Permite ejecutar un camino cuando la condición es verdadera y otro cuando es falsa.

Python

edad = 16 if edad >= 18: print("Mayor de edad") else: print("Menor de edad")

Java


        int edad = 16; 
            if(edad >= 18){
                System.out.println("Mayor de edad");
            }else{
                System.out.println("Menor de edad");
            }

🔁 Ciclo FOR

Sirve para repetir instrucciones una cantidad determinada de veces.

Python

for i in range(5):
    print(i)

Java

for(int i = 0; i < 5; i++){
    System.out.println(i);
}

🔄 Ciclo WHILE

Repite instrucciones mientras una condición sea verdadera.

Python


        contador = 1
            while contador <= 5:
                print(contador)
            contador += 1

Java


        int contador = 1;
            while(contador <= 5){
                System.out.println(contador);
                contador++;
        }

⚙️ Funciones

Una función es un bloque reutilizable de código. Evita repetir instrucciones.

Python

def saludar():
    print("Hola")

saludar()

Java

public static void saludar(){
    System.out.println("Hola");
}

saludar();

📋 Listas / Arrays

Permiten almacenar varios datos dentro de una misma estructura.

Python

equipos = ["Real Madrid", "Barcelona", "Liverpool"]

print(equipos[0])

Java

String[] equipos = {
 "Real Madrid",
 "Barcelona",
 "Liverpool"
};

System.out.println(equipos[0]);

🏛️ Programación Orientada a Objetos (POO)

La POO permite representar objetos del mundo real dentro de un programa. Es uno de los conceptos más utilizados en proyectos empresariales.

Python

class Persona:

    def __init__(self, nombre):
        self.nombre = nombre

persona = Persona("Juan")

print(persona.nombre)

Java

class Persona{

    String nombre;

    Persona(String nombre){
        this.nombre = nombre;
    }
}

⚠️ Manejo de Errores

Permite controlar errores para que la aplicación no se detenga.

Python

try:
    numero = 10 / 0
except:
    print("Ocurrió un error")

Java

try{
    int x = 10 / 0;
}catch(Exception e){
    System.out.println("Error");
}
